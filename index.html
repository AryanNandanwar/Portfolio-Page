EchoAide

doctorscribe-prod

Watch demo

Built a production AI platform for outpatient clinics that generates clinical notes from doctor-patient conversations, reducing documentation time compared to manual note-taking.

  • Engineered a real-time transcription pipeline using Socket.IO and Soniox Streaming STT, reducing note generation latency by 90% compared to the previous batch-processing workflow.
  • Integrated Claude Sonnet (AWS Bedrock) with a few-shot prompting strategy to generate structured clinical notes, improving note completeness.
  • Implemented patient intake queues, historical note retrieval, and role-based access control, reducing consultation setup time.
  • Deployed the platform using Docker, Nginx, and my Ubuntu virtual machine hosted on OCI, enabling secure production usage and zero-downtime releases.
